# 📁 폴더 용량 비교 도구 (`test.py`)

두 폴더(원본 ↔ 복사본)의 **파일 수**와 **전체 용량**을 빠르게 비교하는 간단한 GUI 프로그램입니다.

---

## 실행 방법

```bash
python test.py
```

> Python 3.8 이상 권장

---

## 필요 라이브러리

| 라이브러리 | 용도 | 기본 설치 여부 |
|---|---|---|
| `tkinter` | GUI | ✅ Python 기본 포함 |
| `pandas` | 엑셀 파일 읽기 | ❌ 별도 설치 필요 |
| `openpyxl` | `.xlsx` 파싱 | ❌ 별도 설치 필요 |

```bash
pip install pandas openpyxl
```

> `pandas`가 없어도 실행은 가능하며, 엑셀 불러오기 기능만 비활성화됩니다.

---

## 주요 기능

### 1. 엑셀 파일로 일괄 비교
- `src` / `dest` 열이 있는 `.xlsx` 파일을 불러와 여러 경로를 한꺼번에 비교합니다.

### 2. 직접 입력
- 원본(src)과 복사(dest) 경로를 직접 입력하거나 폴더 선택 버튼으로 지정합니다.

### 3. 결과 확인
- **파일 수** 일치 여부: ✅ / ❌
- **전체 용량** 일치 여부: ✅ / ❌
- 결과는 색상으로 구분됩니다 (초록 = 일치, 빨강 = 불일치).

### 4. CSV 저장
- 비교 결과를 `.csv` 파일로 내보낼 수 있습니다.

---

## 엑셀 파일 형식

| src | dest |
|---|---|
| C:\원본\폴더A | D:\백업\폴더A |
| C:\원본\폴더B | D:\백업\폴더B |

- 열 이름은 반드시 `src`, `dest` 이어야 합니다.
- 경로에 보이지 않는 방향 제어 문자(`\u200e` 등)가 포함된 경우 자동으로 제거합니다.

---

## 참고

- 심볼릭 링크는 따라가지 않습니다.
- 접근 권한이 없는 폴더는 건너뜁니다.
- 비교 작업은 백그라운드 스레드에서 실행되어 UI가 멈추지 않습니다.
